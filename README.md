# Tiny-Trivia

Tiny-Trivia is an android application that asks random questions and lets you choose an answer from either true or false. It also tracks your score and questions. It fetches data using volley API from the internet and saves progress with shared preferences. It also has animation and a modern design to make the app more user-friendly.
